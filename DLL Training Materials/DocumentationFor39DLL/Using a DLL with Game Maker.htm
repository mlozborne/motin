How To

C++ Code for the DLL

#define DLLEXPORT extern "C" __declspec(dllexport)

DLLEXPORT double addem (double a, double b){
    return a + b; 
}

Code that must be executed in the game

theId = external_define("TestDll.dll", "addem", dll_cdecl , ty_real, 2, ty_real,ty_real);
theSum = external_call(theId, 1, 2);

room_caption = string (theSum);

external_free("TestDll.dll");

Further instructions

  • Build an exe file
  • Put TestDll.dll in the same folder as the exe
  • Run the exe in an unrestricted mode (supervisory mode)
