Using a DLL

Also see the instruction in the folder for making a DLL.

Put Testdll.lib in the same folder as the project and source code (need during compilation and linking)

Put Testdll.dll in the same folder as the exe file (usually the debug folder, needed during execution)

Code for the DLL

#define DLLEXPORT extern "C" __declspec(dllexport)

DLLEXPORT double addem (double a, double b){
    return a + b; 
}

Code for the test program (this is a standard console application)

#include <iostream>
using namespace std;
#define DLLEXPORT extern "C" __declspec(dllexport)
DLLEXPORT double addem(double, double);
void main(){
    cout << addem(1, 2) << endl;
}
